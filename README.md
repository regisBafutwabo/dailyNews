# Daily News

Nextjs Project using the new ``app`` directory of Nextjs v13.

For more details on the new nextjs ``app`` directory: [Nextjs app dir](https://beta.nextjs.org/docs/getting-started)

## Getting Started

First, install the dependencies:

```bash 
npm install 
# or
yarn 
```
Second, run it:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
